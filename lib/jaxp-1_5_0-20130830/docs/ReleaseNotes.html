JavaTM API for XML Processing Release Notes

Specification Version: 1.5
Reference Implementation (RI) Version: 1.5.0
August 30, 2013

The JAXP 1.5.0 release corresponds to Java SE 7 update 40. It includes security and conformance enhancement, and fixes of critical issues in the areas of performance, and regression. The sources of this release have been integrated into OpenJDK 7 update 40. Please read below for more details about this update.

Changes and bug fixes:

  1. Security
    JAXP 1.5 adds three new properties along with their corresponding System Properties to allow users to specify the type of external connections that can or can not be permitted. This addition gives controls of fetching external resources to users so that the process can be restricted at users' discretion. For more details, please see JAXP 1.5 guide.

  2. Others
    Other than implementing JAXP 1.5, this release also includes other security, conformance fixes. Please refer to:
